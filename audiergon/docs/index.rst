Welcome to Audiergon's Documentation!
======================================

Welcome to Audiergon! This is a suite of audio manipulation tools, all utilising the Fourier Transform.

To install, use the following command:

.. code-block:: bash

   pip install audiergon

Or, clone the GitHub repository `hamdivazim/Audiergon <https://github.com/hamdivazim/Audiergon>`_ and look into the ``apps/`` directory to try the local Audiergon Lab.
